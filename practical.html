CSS Transitions and Transformation

Transforms - move or change the appearance of an element, and
Transitions - make the element smoothly and gradually change from one state to another.

When used together, these properties allow you to create simple animations and add valuable interaction and visual feedback for your users.

Features

  • countless possibilities
  • really simple to implement.
  • Just a couple of lines of code will give you an awesome effect
  • No need for any external library or plugin
  • increase engagement and conversions.

CSS transforms

With the CSS transform property you can rotate, move, skew, and scale elements. Transforms are triggered when an element changes states, such as on mouse-hover or mouse-click.

translate, scale ,rotate ,skew etc

translate

The translate value moves an element left/right and up/down A positive X value moves the element to the right, while a negative X moves the element to the left. A positive Y value moves the element downwards and a negative Y value, upwards.

.box1:hover { transform: translateX(200px); }

scale

The scale value allows you to increase or decrease the size of an element. For example, the value 2 would transform the size to be 2 times its original size. The value 0.5 would transform the size to be half its original size.

.box2:hover { transform: scale(2); }

rotate

With the rotate value, the element rotates clockwise or counterclockwise by a specified number of degrees. A positive value, such as 90deg, rotates the element clockwise, while a negative value, such as -90deg, rotates it counterclockwise.

.box3:hover { transform: rotate(110deg); }

Skew

Used to skwe a div

.box4:hover { transform: skew(50deg); }

Matrix

Used to matrix a div

.box5:hover { transform: matrix(1, -0.3,0,1,0,0); }

CSS transitions

Without a transition, an element being transformed would change abruptly from one state to another. By applying a transition you can control the change, making it smooth and gradual.

Shorthand syntax for transition is

div {
transition: [property] [duration] [timing-function] [delay];
}
